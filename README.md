# Beyond the Horizon - AI-Powered Space Exploration Game

## Overview
A procedurally generated universe exploration game emphasizing discovery, science, and immersion.

## Project Structure
- UnityProject/: Main game client (C#)
- Backend/: FastAPI backend services
- PythonScripts/: Utility scripts for generation
- Tests/: Unit and integration tests

## Technology Stack
- Unity (LTS)
- C#
- Python FastAPI
- PostgreSQL + Redis
